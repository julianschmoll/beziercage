# jSmear Deformer

Maya Smear deformer.

## Getting Started


```python
from maya import cmds

cmds.jSmear()
```

## Building

### Building on Windows

1. Clone the repository
2. Run `build <MayaVersion>` in root of repository

## Building on Other Systems

1. Clone the repository
2. Run `./build.sh <MayaVersion>`
